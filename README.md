
# Inventory Management System

A full-stack Inventory Management System built with Spring Boot and MySQL.

## Tech Stack

* Java Spring Boot
* MySQL
* REST APIs
* HTML / CSS / JavaScript
* Chart.js
* Swagger API Documentation

## Features

* Product CRUD operations
* Inventory stock tracking
* Automatic stock status classification
* Dashboard analytics
* Category-based insights
* Swagger API documentation

## Backend

Run the backend server:

cd inventory-code/backend
mvn spring-boot:run

Backend runs at:

http://localhost:8080

Swagger documentation:

http://localhost:8080/swagger-ui/index.html

## Frontend

Run frontend using Live Server.

Open:

frontend/index.html

Application runs at:

http://127.0.0.1:5500
